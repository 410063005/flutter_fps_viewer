# flutter_fps_viewer

Flutter FPS Viewer.

## Usage

Show fps in an overlay:

```dart 
showFpsViewer(context);
```

Hide fps overlay if necessary:

```dart
hideFpsViewer();
```

You can also use FpsViewerWidget as an Widget:

```dart
Container(
    child: FpsViewerWidget();
)
```
